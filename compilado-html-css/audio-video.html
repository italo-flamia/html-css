Áudio

A inserções de áudios também é nativa do html, uitiliza-se a tag <audio>:

Parâmetros:

  • preload = indica se o áudio será pré carregado ou não;
    • "metadata" = carrega apenas algumas informações sobre o arquivo (tamanho, tempo, informações de direitos, etc.)
    • "none" = não carrega até que o usuário clique no botão de play ou um script inicie a reprodução.
    • "auto" (padrão) = carrega o arquivo inteiro inteiro assim que a página é carregada, mesmo que o usuário não interaja.
  • controls = o player aparece na tela;
  • autoplay = inicia a reprodução do automaticamente assim que a página é carregada;
  • loop = faz com que o áudio seja repetido automaticamente;
  • <source src="áudio.extensão" type="audio/extensão">

Vídeo

Também podemos inserir vídeos de forma nativa usando a tag <video>:

Parâmetros:

  • width = "valor-da-largura"
  • poster = "thumb.extensão"
  • controls = o player aparece no vídeo
  • autoplay = inicia a reprodução do automaticamente assim que a página é carregada;
  • muted = a opção de sem som vem pré-marcada nos controles;
  • <source src="vídeo.extensão" type="video/extensão">
