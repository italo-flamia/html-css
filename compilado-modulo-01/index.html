A semântica em HTML é intríseca a nova versão ou seja as tags precisam ter um significado, alterações em sua forma ficaram a encargo das CSS.


Links e Âncoras

Links e âncoras fundamentam a navegação da internet, tanto dentro do próprio site quanto para fora dele, são uma das bases para interatividade. Ao clicar em um site, baixar um arquivo, entre outros. Estamos interagindo com um hyperlink. Para criar um link é usada a tag <a>, e ela possui alguns parâmetros listados abaixo:

<a> âncora

Parâmetros:

  • href = "link-de-referência"(interno ou externo)
  • hreflang = linguagem do site direcionado
  • target=
    • "_blank" - abre uma nova janela em branco
    • "_self" - abre na mesma janela (padrão)
  • rel=
    • "next" - indica que o link é para a próxima parte do documento atual;
    • "prev" - indica que o link é para a parte anterior do documento atual;
    • "author" - indica que o link é para o site do autor do artigo atual;
    • "external" - indica que o link é para outro site que não faz parte do site atual;
    • "nofollow" - indica que é um link para um site não endossado, como um link pago;
  • download = "nome-do-arquivo.extensão"
  • type = "tipo-do-arquivo.extensão" (pesquisar)


Imagens Flexíveis

A visualização do site pode ocorrer em diferentes tamanhos de tela, é necessário que o site proporcione a melhor experiência possível independente da tela. Por isso quando exibimos uma imagem podemos adequá-la de acordo com a especificação pré definida. A tag utilizada é a <picture> e possui alguns parâmetros para adequar a exibição a tela.

<picture>

Parâmetros:

  • source media = "definição de largura da tela"
    • min-width = "largura mínima"
    • max-width = "largura máxima"
  • srcset = "fonte da imagem"
  • type = "tipo da imagem"
  • img src = "fonte da imagem padrão"
  • alt = "texto alternativo"
